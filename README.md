# 📊 Sales Data Analysis & Insights

## 🚀 Overview

This project analyzes real-world sales data to uncover revenue trends, customer behavior, and product performance.
The goal is to derive actionable insights that can help businesses optimize sales strategies and improve decision-making.

---

## 🛠️ Tech Stack

* Python (Pandas, NumPy)
* Data Visualization (Matplotlib, Seaborn)
* Jupyter Notebook

---

## 📂 Project Structure

```
sales-data-analysis/
│── data/              # Raw dataset
│── notebooks/         # Analysis notebook
│── images/            # Visualizations
│── README.md
```

---

## 📊 Key Business Questions Solved

* Which month generated the highest revenue?
* What time of day has the highest sales?
* Which products are most frequently sold?
* Which cities contribute the most to revenue?
* What product combinations are commonly bought together?

---

## 📈 Key Insights

* 📌 December recorded the highest sales due to seasonal demand
* 📌 Peak purchase hours: 10–11 AM and 6–8 PM
* 📌 Top-selling products: Batteries, Charging Cables
* 📌 Major revenue contribution comes from metropolitan cities
* 📌 Certain product bundles are frequently purchased together

## ▶️ How to Run

1. Clone the repository
2. Install dependencies:

   ```
   pip install pandas numpy matplotlib seaborn
   ```
3. Open Jupyter Notebook:

   ```
   jupyter notebook
   ```
4. Run `sales_analysis.ipynb`
